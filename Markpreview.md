# What is Markdown?

Markdown is a lightweight markup language that is used to format plain text in a simple and readable way. It was created to make writing documents easier without using complex word-processing software. Instead of clicking formatting buttons, you use simple symbols to create headings, bold text, italic text, lists, links, images, and code blocks. The text remains easy to read even before it is formatted.

## **Advantages of Markdown**

One of the biggest advantages of Markdown is its simplicity and portability. A Markdown file can be opened in almost any text editor and can be converted into HTML, PDF, or other formats. It is widely used by students, developers, technical writers, and content creators because it saves time and keeps documents clean and organized. Platforms like GitHub, GitLab, and many blogging websites support Markdown.

## **Markdown Cheat Sheet** 🧾

## **Symbol vs Purpose**

| Symbol / Syntax | Purpose 🎯 |
|----------------|-----------|
| `#` | Main heading (H1) |
| `##` | Section heading (H2) |
| `###` | Sub-heading (H3) |
| `**text**` | Bold text |
| `*text*` | Italic text |
| `***text***` | Bold + Italic |
| `~~text~~` | Strikethrough |
| `-` | Unordered list item |
| `1.` | Ordered list item |
| `[text](url)` | Create hyperlink |
| `![alt](img)` | Insert image |
| `` `code` `` | Inline code |
| ``` ``` | Code block |
| `>` | Blockquote (quote/note) |
| `---` | Horizontal line |
| `- [ ]` | Task (unchecked) |
| `- [x]` | Task (checked) |
